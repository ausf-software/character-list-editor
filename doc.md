### Загрузка нового пакета правил

1. Запустите приложение.
2. На главном экране нажмите кнопку **«Установленные пакеты»**.
3. В открывшемся окне нажмите кнопку **«Добавить пакет»**.
4. Выберите файл с пакетом правил в стандартном диалоге выбора файлов.
5. После выбора файла пакет автоматически установится и появится в общем списке установленных пакетов.
6. Вы можете закрыть окно «Установленные пакеты» или продолжить работу со списком.

### Удаление пакета правил

Если пакет правил больше не нужен, его можно удалить.

1. Запустите приложение.
2. Нажмите кнопку **«Установленные пакеты»**.
3. В списке пакетов отметьте галочкой (чекбокс) те пакеты, которые хотите удалить. Кнопка **«Удалить»** станет активной.
4. Нажмите кнопку **«Удалить»**.
5. Появится диалоговое окно с запросом подтверждения, например: *«Вы действительно хотите удалить выбранные пакеты (1 шт.)?»*.
6. Если вы уверены в удалении, нажмите **«Да»**. Если передумали — нажмите **«Нет»** и удаление не произойдёт.
7. После подтверждения выбранные пакеты исчезнут из списка, а кнопка «Удалить» снова станет неактивной.
8. Закройте окно «Установленные пакеты».

### Добавление тега персонажу

Теги помогают классифицировать персонажей (например, «боевой», «маг», «лидер»). У одного персонажа может быть несколько тегов, но все они должны быть уникальны в рамках этого персонажа.

1. Запустите приложение.
2. На главном экране выберите нужного персонажа и нажмите кнопку **«+ Тег»** (или аналогичный элемент добавления тега).
3. Откроется окно создания тега. В нём доступны:
   - поле для ввода названия;
   - кнопка выбора цвета;
   - область предпросмотра (отображает текущий вид тега);
   - кнопки **«Добавить»** и **«Отмена»**.
4. Введите название тега (например, «боевой»). Поле не должно быть пустым.
5. При необходимости нажмите на кнопку выбора цвета и выберите нужный цвет из палитры. Подтвердите выбор.
6. В области предпросмотра отобразится название тега на выбранном фоне.
7. Чтобы добавить тег, нажмите **«Добавить»**. Окно закроется, и новый тег появится у персонажа.
8. Если вы передумали создавать тег, нажмите **«Отмена»** — никаких изменений не произойдёт.

> **Обратите внимание:** нельзя добавить два одинаковых тега одному персонажу. Если вы попытаетесь создать тег с названием, которое уже есть у этого персонажа, новый тег не добавится.

### Удаление персонажа из списка

Вы можете удалить ненужного персонажа из общего списка. Перед удалением система запросит подтверждение.

1. Запустите приложение.
2. На главном экране найдите персонажа, которого хотите удалить (например, «Герой 1»).
3. Справа от имени персонажа нажмите кнопку **«Удалить»** (обычно обозначена значком корзины).
4. Появится модальное окно с вопросом, например: *«Вы действительно хотите удалить персонажа "Герой 1"?»*.
5. Если вы передумали, нажмите **«Нет»** — окно закроется, персонаж останется в списке.
6. Для окончательного удаления нажмите **«Да»**.
7. Персонаж исчезнет из списка. Остальные персонажи сохранятся.
8. Вы можете продолжить работу или закрыть приложение. Удалённый персонаж не восстановится после перезапуска.

### Создание нового персонажа

Чтобы добавить нового персонажа, используйте мастер создания, где можно выбрать пакеты правил, шаблон листа и указать кампанию.

1. Запустите приложение.
2. На главном экране со списком персонажей нажмите кнопку **«Создать»** (или **«Новый персонаж»**).
3. Откроется диалоговое окно **«Создание персонажа»**, содержащее:
   - список установленных пакетов правил с чекбоксами для выбора;
   - выпадающий список для выбора шаблона листа персонажа;
   - текстовое поле **«Кампания»**;
   - кнопки **«Создать»** и **«Отмена»**.
4. Отметьте галочками один или несколько пакетов, которые будут использоваться для персонажа (например, «Базовые правила», «Магия»).
5. Раскройте выпадающий список и выберите подходящий шаблон (например, «Шаблон воина»).
6. В поле **«Кампания»** введите название кампании, в которой участвует персонаж (необязательно, но рекомендуется).
7. Когда все необходимые поля заполнены, кнопка **«Создать»** станет активной. Нажмите её.
8. Диалог закроется, и автоматически откроется окно **«Просмотр листа персонажа»** для только что созданного персонажа. Просмотрите его характеристики — они соответствуют выбранному шаблону и пакетам.
9. Закройте окно просмотра (кнопкой **«Назад»** или **«Закрыть»**).
10. Новый персонаж появится в общем списке на главном экране. Вы можете открыть его снова, чтобы убедиться, что все данные сохранены.

> **Если вы передумали создавать персонажа:** в диалоге создания нажмите кнопку **«Отмена»**. Персонаж не будет создан, список останется без изменений.
